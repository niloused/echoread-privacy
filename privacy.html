Privacy Policy — EchoRead

Effective Date: July 2025

1. Introduction

EchoRead is a Chrome extension that helps users convert web articles, PDFs, and online content into natural, human-like audio using advanced AI technologies. We prioritize your privacy and are committed to keeping your data safe and secure.

2. Information Collection

EchoRead does not collect or store any personal information:

  • We do not collect your browsing history.
  • We do not collect or store user credentials.
  • We do not collect personally identifiable information (PII).

The extension only processes the text content from the web pages or files that you choose to process via the extension features.

3. Data Processing

All processing (summarization, translation, text extraction, text-to-speech) is performed securely via encrypted API calls to third-party services such as:

  • Google Text-to-Speech API
  • Hugging Face Inference API
  • AWS SageMaker (for translation)

No raw personal data is transmitted or stored. Processed data is temporarily held in memory only during the active session and is not retained after processing is completed.

4. Storage of Data

EchoRead does not store user data on our servers:

  • No audio files, transcripts, or processing results are permanently saved.
  • No analytics, tracking, or behavioral data is collected.

5. Permissions

EchoRead requests the following permissions to function:

  • Access to active tab (for extracting article text)
  • Storage permission (to store extension preferences locally)

These permissions are only used to enable the features of the extension and are not used for any unauthorized access.

6. Third-Party Services

While EchoRead uses external APIs for processing, we do not share any personal user information with these services. All API requests are securely transmitted over HTTPS.

7. Changes to This Policy

We may update this privacy policy occasionally to reflect updates to the extension or to comply with legal obligations. Changes will be posted in the Chrome Web Store listing.

8. Contact

For any privacy-related questions or concerns, please contact us: support@echoread.io or visit https://echoread.io
